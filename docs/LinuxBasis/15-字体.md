# {{ $frontmatter.title }}

如果你用 Linux 在 `.docx` 文件中码字，会发现很多字体是没有的。

## 添加与Windows一样的字体
以下方法二选一：
1. 安装`ttf-mscorefonts-installer`包。
    ```bash
    sudo apt install ttf-mscorefonts-installer
    ```
    这是网上较为主流的方法，我试过，并不管用。
2. 将Windows中的`C:\Windows\Fonts`目录下的所有文件复制到Linux中的`/usr/share/fonts/`目录或者子目录（最好是这个，名为`msfonts`）下。

然后，执行`fc-cache`命令，刷新字体缓存即可。
